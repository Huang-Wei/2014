## Dependencies

1. node
2. mongodb

## How to run this project

Download:
```
git clone git@github.com:Huang-Wei/2014.git
```

Install necessary npm packages:
```
cd 2014 && npm install
```

Start:
```
node app.js
```

## DB init

Make sure `mongod` is running, run `db/init.js` to initilaize the db
```
mongo 2014 db/init.js
```
